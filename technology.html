How to Resize an image in Pixlr
Clcik where you would like to source the image from to start
Click Image bar > Image Size
Select the size, which will be from 4000x3000 (original) to 533x400 (new)

How to Change an image filetype in Pixlr

Clcik where you would like to source the image from to start
Click File --> Save....
You will then get the 'Save' window
You can then select the output File Type
